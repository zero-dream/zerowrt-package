  ### ZeroWrt-Package

  ZeroWrt (Immortalwrt) 固件专用软件包

  支持系列: QUALCOMMAX, MEDIATEK, ROCKCHIP, X86

  ### 参考来源

  [VIKINGYFY/OpenWRT-CI](https://github.com/VIKINGYFY/OpenWRT-CI)

  \--------------------------------------------------

  ### 固件下载

  [zero-dream/zerowrt-firmware](https://github.com/zero-dream/zerowrt-firmware)

  \--------------------------------------------------

  ### Fork 注意事项

  本项目包含闭源程序，无法正确执行工作流

  ### 目录简要说明

  固件文件名的时间为开始编译的时间，方便核对上游源码的提交时间

  | 目录 | 作用 |
  | :---: | :---: |
  | zerodream | 私有目录，禁止修改 |
  | hook | Workflow 专用目录 |
  | application | 应用目录 |
  | config | 配置目录 |
  | script | 脚本目录 |
  | storage | 储存目录 |
